SCRABBLE PROGRAM IN C++
By Jacob Moors

1. Open folder in Ubuntu environment (WSL for windows)
2. Remove any previous files from the makefile using: make clean
3. Run the following command to create the files: make all
4. Now run the scrabble program using: ./scrabble
5. If you require assistance during the game, type 'help'
6. To place a tile, use the format 'place A at B4'
7. Once you've finished placing the tiles, use 'place done' and the program will check if the word is valid 
8. Type 'save filename' to save the game in its current state, which can be loaded from the main menu
